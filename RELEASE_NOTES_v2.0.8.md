# CarExplorer v2.0.8 Release Notes

## What's New
- **Improved Reliability**: Enhanced stability for fetching car information from Wikipedia and Wikidata sources. The app now handles API responses more robustly, ensuring you get car details even under varied network conditions.
- **Better Error Handling**: More graceful handling of API errors and timeouts, with improved logging to help diagnose issues.

## Fixes
- Fixed reliability issues when fetching car specifications and images
- Improved consistency in API error reporting
- Enhanced mock testing infrastructure to ensure API responses are handled correctly

## Performance
- Faster response times when retrieving car details
- More efficient API error recovery

## Known Issues
- Navigation responsiveness on initial app launch is being investigated. This does not affect core functionality.

## Requirements
- Android 8.0 (API 26) or higher

# CarExplorer v2.0.8 - Notatki wydania

## Co nowego
- **Lepsze niezawodność**: Poprawiona stabilność pobierania informacji o samochodach ze źródeł Wikipedia i Wikidata. Aplikacja teraz bardziej niezawodnie obsługuje odpowiedzi API, zapewniając szczegóły samochodu nawet w różnych warunkach sieciowych.
- **Lepsze obsługiwanie błędów**: Bardziej wdzięczna obsługa błędów API i limitów czasu, z ulepszonymi logami diagnostycznymi.

## Poprawki
- Naprawiono problemy niezawodności podczas pobierania specyfikacji i obrazów samochodów
- Poprawiona spójność raportowania błędów API
- Ulepszona infrastruktura testowania zapewniająca prawidłową obsługę odpowiedzi API

## Wydajność
- Szybsze czasy odpowiedzi podczas pobierania szczegółów samochodu
- Bardziej efektywne odzyskiwanie po błędach API

## Znane problemy
- Responsywność nawigacji podczas pierwszego uruchamiania aplikacji jest badana. Nie wpływa to na podstawową funkcjonalność.

## Wymagania
- Android 8.0 (API 26) lub nowszy
